Estilo de letra

head o encabezado

Etiqueta Head: la etiqueta head, conocida en español como cabecera, sirve para contener varios "hijos que otorgan información en cuestión, pero esta etiqueta no se visualiza en la página. sirve para conectar el CSS o cualquier tipo de metadato. estos son los elemento que puede contener el head. title (obligatorio en todos los documentos HTML) style> base> link> meta> script> noscript>

body o cuerpo

este elemento define el cuerpo del documento y en un contenedor de contenidos visibles como encabezados, párrafos, imágenes, hipervínculos, tablas, listas, etc. elemento title:

titulo o "title"

este elemento sirve para colocar un título o nombre al documento, el cual debe ser corto y y conciso, este da una descripción al documento o su propósito aún cuando es sacada de contexto (como, por ejemplo, cuando se muestra en los resultados de una búsqueda).

Elemento "heading"

el elemento h(heading o encabezado) tiene 6 niveles(h1-h6), donde el h1 es el más importante. Un elemento de encabezado describe brevemente el tema de la sección que presenta. este elemento va en el body.
